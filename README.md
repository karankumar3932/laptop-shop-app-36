# 💻 LaptopHub – Laptop Shop App

## 📌 Project Overview

LaptopHub is a modern **e-commerce web application** developed for browsing and purchasing laptops online.
The app provides a smooth user experience with product listings, authentication, and responsive UI.

---

## 🚀 Features

* 🛒 Browse laptop products
* 🔍 View product details
* 👤 User authentication (Login/Signup)
* 📦 Order management
* 💳 Add to cart functionality
* 📱 Responsive design

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* Vite
* TypeScript
* Tailwind CSS / ShadCN UI

**Backend & Database:**

* Supabase (Authentication + Database)

**Other Tools:**

* Git & GitHub
* Android Studio (for integration/testing)

---

## 📂 Project Structure

```
laptop-shop-app-36/
│── public/          # Static files
│── src/             # Main source code
│── supabase/        # Backend config
│── android/         # Android integration
│── package.json     # Dependencies
```

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/karankumar3932/laptop-shop-app-36.git
```

### 2️⃣ Navigate to project folder

```bash
cd laptop-shop-app-36
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the project

```bash
npm run dev
```
